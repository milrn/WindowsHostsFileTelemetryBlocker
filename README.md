# WindowsHostsFileTelemetryBlocker
Short python script to help manage hosts file list for blocking known Windows telemetry.
# What should I contribute?
- Bug Fixes
- New Auto-Updated Repositories
- New Install/Uninstall Options
- Menu Changes
- Docs
# What constitutes an issue?
- Bugs
- Auto-Updated Repository Gets Maliciously Infected
- Auto-Updated Repository Gets a New Format of Entries
- Feature Requests
# Basic Usage
- Make Sure You Have Python Installed
- Launch an Administrator Terminal on Windows 11
- Run the Script: python {file path to blocker.py} (Make Sure the Script is NOT in a Protected Folder)
- Select the Option You Want
